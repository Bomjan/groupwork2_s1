A clean and modern web interface for **Innovate Green AI**, showcasing the main page, profile, and dashboard. Built with HTML, CSS, and JavaScript with a green/white aesthetic.

## Features

- **Modern Design**: Clean green/white theme with smooth hover effects.
- **Responsive Layout**: Works well on desktop, tablet, and mobile devices.
- **Sections**:
  - Main page: Overview and introduction.
  - Profile: Detailed profile information.
  - Dashboard: Admin-like interface layout.
- **Interactive Elements**: Subtle zoom effects on images for a modern feel.
- **Navigation**: Sticky header with links to sections.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome for icons
